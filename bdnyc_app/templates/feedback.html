AstrodbWeb & BDNYC Database Feedback

Have a suggestion for improving this tool? A correction on a particular object or set of objects?
Have you found a potential bug or error in our website? Just want to ask us a question? Let us know!

If you use GitHub, you can always submit new Issues to the BDNYC database repository or
the AstrodbWeb repository, depending on the nature of your comment.

Alternatively, you can use the form below and we'll address it as best as we can.
Please try to be as descriptive as possible about any errors; such as the query to reproduce your error or the source name/id number with missing/incorrect data.
